# tietoliikenteen-perusteet

Kurssilla tutustutaan tietoliikenteen perusteisiin. Siis miten digitaalista (bittejä) tietoa siirretään johtoa ja ilmatietä pitkin. Samalla, kun näitä perusteita käydään läpi tehdään mittauksia oskilloskooppia ja spektrianalysaattoria hyväksi käyttäen.

Kurssin "toisessa osassa"  tutustutaan matkapuhelinverkon toimintaperiaatteeseen ja protokollapinoon 5G matkapuhelinverkkoa esimerkkinä käyttäen.

Kurssin viimeisessä osassa valmistaudutaan tulevaan tietoliikenteen sovellusprojektiin rakentamalla kaksisuuntainen langaton yhteys 433 MHz ASK radiopiirejä käyttäen.
